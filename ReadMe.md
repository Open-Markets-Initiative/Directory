## The Open Markets Initiative

[The Open Markets Initiative](https://github.com/Open-Markets-Initiative/Directory/tree/main/About "About Omi") (Omi) is dedicated to enhancing the stability of electronic financial markets using modern development methods.

This directory contains public protocol specifications, sample data references, and documentation for 33 financial market organizations.

All documentation has been collected from publicly available sources. These documents also preserve the historical record as organizations often remove inactive protocols.
