## Iex Protocols

Protocol specifications for Investors Exchange market data and transport. Iex provides real-time equity market data over its proprietary IexTp transport protocol using Udp multicast delivery.

| Protocol | Type | Description |
| --- | --- | --- |
| [Deep](Deep.md) | Market Data | Full depth of book market data |
| [Tops](Tops.md) | Market Data | Top of book quotes and last sale |
| [IexTp](IexTp.md) | Transport | Iex Transport Protocol |

## Specifications

Protocol specifications for each protocol are available in the [Specifications](../Specifications) directory.
