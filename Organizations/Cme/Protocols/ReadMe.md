## Cme Group Protocols

Protocol specifications for Cme Group exchanges (Cme, Cbot, Nymex, Comex). All protocols use Simple Binary Encoding (Sbe) with Fix 5.0 Sp2 semantics and little-endian byte ordering.

| Protocol | Type | Description |
| --- | --- | --- |
| [iLink3](iLink3.md) | Order Entry | Cme Globex order entry |
| [Mdp3](Mdp3.md) | Market Data | Market Data Platform 3.0 |
| [Streamlined](Streamlined.md) | Market Data | Streamlined market data |
| [Settlements](Settlements.md) | Market Data | Settlement price data |
| [SessionMgmt](SessionMgmt.md) | Session | Market data session management |
| [BrokerTec](BrokerTec.md) | Market Data | BrokerTec Us Treasury market data |
| [EbsSpectrum](EbsSpectrum.md) | Market Data | Ebs Fx derived market data |

## Specifications

Sbe Xml schema templates for each protocol are available in the [Specifications](../Specifications) directory.
