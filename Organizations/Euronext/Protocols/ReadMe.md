## Euronext Protocols

Protocol specifications for Euronext exchange connectivity using the Optiq platform with Simple Binary Encoding (Sbe) for market data, order entry, and post-trade services.

| Protocol | Type | Description |
| --- | --- | --- |
| [MarketDataGateway](MarketDataGateway.md) | Market Data | Optiq Sbe market data feed |
| [OrderEntryGateway](OrderEntryGateway.md) | Order Entry | Optiq Sbe order entry |
| [DropCopy](DropCopy.md) | Post-Trade | Optiq Sbe drop copy for execution reporting |

## Specifications

Sbe schema definitions for each protocol are available in the [Specifications](../Specifications) directory.
