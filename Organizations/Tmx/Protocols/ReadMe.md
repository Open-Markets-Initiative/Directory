## Tmx Protocols

Protocol specifications for Tmx Group exchanges including Toronto Stock Exchange (Tsx), Tsx Venture Exchange (Tsxv), Tsx Alpha Exchange, and Montreal Exchange (Mx).

| Protocol | Type | Description |
| --- | --- | --- |
| [Hsvf](Hsvf.md) | Market Data | High Speed Vendor Feed for Mx derivatives |
| [Sail](Sail.md) | Order Entry | Sola Access Information Language for Mx derivatives |
| [Xmt](Xmt.md) | Market Data | eXtreme Message Transfer for Tsx/Tsxv Level 2 |
| [QuantumFeed](QuantumFeed.md) | Market Data | Tsx/Tsxv/Alpha Level 1 and Level 2 market data |
| [Stamp](Stamp.md) | Order Entry | Tsx/Tsxv/Alpha order entry |
| [BroadcastFeed](BroadcastFeed.md) | Market Data | Tmx Ip broadcast feed |
| [Tms](Tms.md) | Trade Management | Sola Trade Management System for Mx |
