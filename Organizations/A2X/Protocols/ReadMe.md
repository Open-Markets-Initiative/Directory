## A2X Protocols

Protocol specifications for the A2X Markets exchange in South Africa. A2X operates on Aquis technology, using binary protocols for market data dissemination.

| Protocol | Type | Description |
| --- | --- | --- |
| [Amd](Amd.md) | Market Data | Aquis market data feed |
