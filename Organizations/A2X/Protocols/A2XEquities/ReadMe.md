## A2X Equities


A2X Markets is a licensed South African stock exchange providing a secondary listing venue for equities, running on Aquis Exchange matching engine and market data technology.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [OrderEntry](OrderEntry.md) | OrderEntry | A2X Markets Fix 4.2 Order Entry and Trade Reporting |
| [Rtmdf](Rtmdf.md) | MarketData | A2X Markets real time market data |
| [Snapshot](Snapshot.md) | MarketData | A2X Markets snapshot market data |
