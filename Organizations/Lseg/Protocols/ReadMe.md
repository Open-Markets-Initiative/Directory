## Lseg Protocols

Protocol specifications for London Stock Exchange Group market data dissemination. Lseg provides market data for venues including the London Stock Exchange, Turquoise, and TradeEcho.

| Protocol | Type | Description |
| --- | --- | --- |
| [Gtp](Gtp.md) | Market Data | Group Ticker Plant |
| [Mitch](Mitch.md) | Market Data | Millennium Itch Level 2 |

## Specifications

Protocol specifications for each protocol are available in the [Specifications](../Specifications) directory.
