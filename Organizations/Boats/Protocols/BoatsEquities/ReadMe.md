## Boats Equities


Blue Ocean ATS US equities alternative trading system operating overnight sessions for trading US-listed securities during Asia and Europe business hours.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [DropCopy](DropCopy.md) | DropCopy | Boats Fix Drop Copy (MEMO FIX Drop) |
| [Memo](Memo.md) | OrderEntry | Boats Sbe Order Entry |
| [MemoirDepthFeed](MemoirDepthFeed.md) | MarketData | Boats Depth Of Book Data |
| [MemoirLastSale](MemoirLastSale.md) | MarketData | Boats Last Sale Trade Data |
| [MemoirTopOfBook](MemoirTopOfBook.md) | MarketData | Boats Best Bid And Offer Data |
| [OrderEntry](OrderEntry.md) | OrderEntry | Boats Fix Order Entry (MEMO FIX) |
