## B3 Protocols

Protocol specifications for B3 (Brasil, Bolsa, Balcao), the Brazilian exchange. B3 uses Sbe-encoded binary protocols with Fixp session semantics for order entry and market data dissemination.

| Protocol | Type | Description |
| --- | --- | --- |
| [BinaryEntryPoint](BinaryEntryPoint.md) | Order Entry | Sbe/Fixp binary order entry |
| [BinaryUmdf](BinaryUmdf.md) | Market Data | Sbe binary market data |
