## Currenex Protocols

Protocol specifications for Currenex Fx trading platforms using Cbp (Currenex Binary Protocol) based messaging for market data and order entry.

| Protocol | Type | Description |
| --- | --- | --- |
| [Esp](Esp.md) | Market Data | Fx streaming prices via Cbp/Itch-based encoding |
| [Now](Now.md) | Market Data | Fx market data via Cbp |
| [OrderService](OrderService.md) | Order Entry | Fx order entry via Cbp/Ouch-based encoding |

## Specifications

Protocol definitions for each interface are available in the [Specifications](../Specifications) directory.
