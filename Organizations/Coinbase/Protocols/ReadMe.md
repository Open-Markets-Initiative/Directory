## Coinbase Protocols

Protocol specifications for Coinbase institutional exchange connectivity using Simple Binary Encoding (Sbe) over dedicated binary interfaces.

| Protocol | Type | Description |
| --- | --- | --- |
| [MarketDataApi](MarketDataApi.md) | Market Data | Sbe market data feed |
| [OrdersApi](OrdersApi.md) | Order Entry | Sbe order entry |
| [Session](Session.md) | Session | Sbe session management |

## Specifications

Sbe schema definitions for each protocol are available in the [Specifications](../Specifications) directory.
