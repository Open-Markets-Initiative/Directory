## Electric Derivatives


ElectronX exchange trading bounded futures and binary options via a FIX 5.0 SP2 order entry gateway (NLS).


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [MarketData](MarketData.md) | MarketData | ElectronX Fix 5.0 Market Data |
| [OrderEntry](OrderEntry.md) | OrderEntry | ElectronX Fix 5.0 Order Entry (NLS) |
