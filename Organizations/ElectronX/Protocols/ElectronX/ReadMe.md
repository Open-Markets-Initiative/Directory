## ElectronX


ElectronX electronic exchange offering Bounded Futures and Binary Options via two dedicated FIX 5.0 SP2 gateways (NLS).


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [MarketData](MarketData.md) | MarketData | ElectronX NLS Fix Market Data |
| [OrderEntry](OrderEntry.md) | OrderEntry | ElectronX NLS Fix Order Entry |
