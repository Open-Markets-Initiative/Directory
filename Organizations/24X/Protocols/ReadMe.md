## 24X Protocols

Protocol specifications for the 24X National Securities Exchange. 24X operates on the Memx technology platform, using Sbe-encoded binary protocols for order entry and market data.

| Protocol | Type | Description |
| --- | --- | --- |
| [Memo](Memo.md) | Order Entry | Sbe binary order entry |
| [MemoirDepthFeed](MemoirDepthFeed.md) | Market Data | Full depth of book market data |
| [MemoirLastSale](MemoirLastSale.md) | Market Data | Last sale trade data |
| [MemoirTopOfBook](MemoirTopOfBook.md) | Market Data | Top of book market data |

## Specifications

Protocol schema definitions are available in the [Specifications](../Specifications) directory.
