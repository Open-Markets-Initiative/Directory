## 24X Equities


24X National Exchange US listed equities and ETFs venue designed to operate around the clock including overnight trading outside of traditional US market hours.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [DropCopy](DropCopy.md) | DropCopy | 24X Fix Drop Copy (MEMO FIX Drop) |
| [Memo](Memo.md) | OrderEntry | 24X Sbe Order Entry |
| [MemoirDepthFeed](MemoirDepthFeed.md) | MarketData | 24X Depth Of Book Data |
| [MemoirLastSale](MemoirLastSale.md) | MarketData | 24X Last Sale Trade Data |
| [MemoirTopOfBook](MemoirTopOfBook.md) | MarketData | 24X Best Bid And Offer Data |
| [OrderEntry](OrderEntry.md) | OrderEntry | 24X Fix Order Entry (MEMO FIX) |
