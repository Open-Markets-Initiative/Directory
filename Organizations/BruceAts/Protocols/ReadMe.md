## BruceAts Protocols

Protocol specifications for Bruce Ats, an alternative trading system. BruceAts uses the Itch binary protocol for equities market data across BestBidAndOffer, DepthOfBook, and LastSale feeds.

| Protocol | Type | Description |
| --- | --- | --- |
| [BestBidAndOffer](BestBidAndOffer.md) | Market Data | Equities Bbo Itch feed |
| [DepthOfBook](DepthOfBook.md) | Market Data | Equities depth of book Itch feed |
| [LastSale](LastSale.md) | Market Data | Equities last sale Itch feed |

## Specifications

Protocol schema definitions are available in the [Specifications](../Specifications) directory.
