## Siac Protocols

Protocol specifications for Securities Industry Automation Corporation (Siac) consolidated market data feeds including the Consolidated Quotation System, Consolidated Tape System, and Options Price Reporting Authority.

| Protocol | Type | Description |
| --- | --- | --- |
| [Cqs](Cqs.md) | Market Data | Consolidated Quotation System best bid and offer quotes |
| [Cts](Cts.md) | Market Data | Consolidated Tape System last sale trade reports |
| [Opra](Opra.md) | Market Data | Options Price Reporting Authority options market data |

## Specifications

Protocol schema definitions are available in the [Specifications](../Specifications) directory.
