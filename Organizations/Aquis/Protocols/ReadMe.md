## Aquis Protocols

Protocol specifications for Aquis Exchange. Aquis provides binary and Fix protocols for market data dissemination and order entry across its European equities trading venues.

| Protocol | Type | Description |
| --- | --- | --- |
| [Amd](Amd.md) | Market Data | Aquis market data feed |
| [Atp](Atp.md) | Order Entry | Aquis trading protocol |
| [Fix](Fix.md) | Order Entry | Fix order entry |

Protocol schema definitions are available in the [Specifications](../Specifications) directory.
