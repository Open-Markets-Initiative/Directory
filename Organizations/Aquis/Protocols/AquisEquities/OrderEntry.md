## AquisEquities Order Entry: Aquis Exchange Aquis FIX 4.2 Technical Specification v5.0

Fix 4.2 order entry protocol for the Aquis Exchange pan-European MTF, providing members and vendors with order entry, cancel/replace, execution reports, and Fix 4.4 trade capture reporting for on-exchange crosses.

### Transport

Tcp for authenticated Fix 4.2 sessions with sequence-number gap recovery via Resend Request and Sequence Reset.
