## AquisEquities Drop Copy: Aquis Exchange Aquis Drop Copy FIX 4.2 Specification v2.1

Fix 4.2 drop copy feed for the Aquis Exchange MTF, delivering a read-only copy of execution reports and trade capture report acknowledgements over an authenticated Fix session.

### Transport

Tcp for authenticated Fix 4.2 sessions with sequence-number gap recovery via Resend Request and Sequence Reset.
