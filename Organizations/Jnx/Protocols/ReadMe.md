## Jnx Protocols

Protocol specifications for Japannext Pts market data and order entry. Jnx operates a proprietary trading system for Japanese equities and bonds.

| Protocol | Type | Description |
| --- | --- | --- |
| [Equities Itch](Equities.Itch.md) | Market Data | Pts equities Itch market data |
| [Equities Ouch](Equities.Ouch.md) | Order Entry | Pts equities Ouch order entry |
| [Bonds Itch](Bonds.Itch.md) | Market Data | Pts bonds Itch market data |
| [Bonds Ouch](Bonds.Ouch.md) | Order Entry | Pts bonds Ouch order entry |

## Specifications

Protocol specifications for each protocol are available in the [Specifications](../Specifications) directory.
