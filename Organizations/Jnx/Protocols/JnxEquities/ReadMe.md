## Japannext Equities


Japannext Proprietary Trading System for equities providing an alternative venue for trading Japanese listed stocks with lower latency and extended hours.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [Itch](Itch.md) | MarketData | Japannext Pts Equities Order By Order Market Data |
| [Ouch](Ouch.md) | OrderEntry | Japannext Pts Equities Order Entry |
