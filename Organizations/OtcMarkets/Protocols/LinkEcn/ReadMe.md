## OTC Link ECN


OTC Link ECN is OTC Markets Group's SEC registered alternative trading system providing anonymous electronic order matching for OTC equity securities. Subscribers enter firm, automatically executable orders over a FIX 4.2 interface across pre-market, regular, and post-market daytime sessions.
