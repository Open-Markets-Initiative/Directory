## MOON ATS


MOON ATS (Markets Open Overnight) is OTC Markets Group's SEC registered alternative trading system for extended overnight trading of NMS-listed US equity securities. Sessions run 18:00 to 04:00 Eastern. Publishes the order-based MOON ATS Overnight Multicast Data Feeds (Otc Link encoding).


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [DepthOfBook](DepthOfBook.md) | MarketData | Order-by-order full depth channel for MOON ATS overnight trading |
| [OrderEntry](OrderEntry.md) | OrderEntry | MOON ATS FIX order entry for overnight NMS trading |
| [Retransmission](Retransmission.md) | MarketData | TCP binary retransmission service for MOON ATS overnight trading recovery |
| [TopOfBook](TopOfBook.md) | MarketData | Aggregated top-of-book channel for MOON ATS overnight trading |
