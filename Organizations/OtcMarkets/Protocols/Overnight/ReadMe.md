## OTC Link Overnight OTC


OTC Link Overnight OTC is OTC Markets Group's overnight quotation platform for OTC equities, running an Inter-dealer Quotation System from 18:00 Eastern to 07:45 Eastern. Order Acceptance at 7:00 PM, Market Open at 8:00 PM, Market Close at 7:45 AM. Publishes the order-based OTC Link Overnight OTC Multicast Data Feeds (Otc Link encoding); shares wire vocabulary with MOON ATS and OTC Link NQB.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [DepthOfBook](DepthOfBook.md) | MarketData | Order-by-order full depth channel for OTC Link Overnight OTC |
| [Retransmission](Retransmission.md) | MarketData | TCP binary retransmission service for OTC Link Overnight OTC recovery |
| [TopOfBook](TopOfBook.md) | MarketData | Aggregated top-of-book channel for OTC Link Overnight OTC |
