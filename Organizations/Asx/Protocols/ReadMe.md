## Asx Protocols

Protocol specifications for the Australian Securities Exchange. Asx uses Itch and Ouch binary protocols across its Ntp, T24, and Trade platforms for market data and order entry.

| Protocol | Type | Description |
| --- | --- | --- |
| [Ntp](Ntp.md) | Market Data | Ntp Itch market data for equities and warrants |
| [T24](T24.md) | Market Data | T24 Itch market data for interest rate derivatives |
| [Trade Itch](Trade.Itch.md) | Market Data | Trade Itch market data for options and futures |
| [Trade Ouch](Trade.Ouch.md) | Order Entry | Trade Ouch order entry for options and futures |
