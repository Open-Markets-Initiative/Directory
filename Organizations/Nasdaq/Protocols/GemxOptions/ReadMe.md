## Nasdaq GEMX


Nasdaq Gemx Options US options exchange operating a pro rata market model within the Nasdaq options complex.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [DepthOfMarket](DepthOfMarket.md) | MarketData | Nasdaq Gemx Options Depth Of Book Data |
| [OrderEntry](OrderEntry.md) | OrderEntry | Nasdaq Nasdaq GEMX Fix order entry |
| [OrderFeed](OrderFeed.md) | MarketData | Nasdaq Gemx Options Order Level Market Data |
| [TopOfMarket](TopOfMarket.md) | MarketData | Nasdaq Gemx Options Best Bid And Offer Data |
| [TradeFeed](TradeFeed.md) | MarketData | Nasdaq Gemx Options Trade Report Data |
