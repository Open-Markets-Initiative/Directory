## Memx Protocols

Protocol specifications for Members Exchange (Memx) equities and options trading. All protocols use Simple Binary Encoding (Sbe) with a common header framework for Tcp and Udp transport.

| Protocol | Type | Description |
| --- | --- | --- |
| [Memo](Memo.md) | Order Entry | Binary order entry for equities and options |
| [MemoirDepth](MemoirDepth.md) | Market Data | Full depth of book for equities and options |
| [MemoirTop](MemoirTop.md) | Market Data | Top of book for equities and options |
| [MemoirLastSale](MemoirLastSale.md) | Market Data | Trade feed for equities |
| [RiskControl](RiskControl.md) | Risk | Options risk controls |
| [CommonHeader](CommonHeader.md) | Transport | Tcp and Udp session headers |

## Specifications
