## T7


T7 is Deutsche Boerse Group's trading platform serving the Eurex derivatives market and the cash markets, providing the Eti and Xti order entry interfaces, the Eobi order book feed, Emdi/Mdi/Rdi market and reference data interfaces, Edci clearing data, and FIX LF gateways.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [CashOrderEntry](CashOrderEntry.md) | OrderEntry | Eurex Eurex Cash Fix order entry |
| [Edci](Edci.md) | MarketData | Eurex T7 Clearing Reference Data |
| [Eobi](Eobi.md) | MarketData | Eurex T7 Order Book Market Data |
| [Eti](Eti.md) | OrderEntry | Eurex T7 Order Entry |
| [OrderEntry](OrderEntry.md) | OrderEntry | Eurex Eurex Derivatives Fix order entry |
| [Xti](Xti.md) | OrderEntry | Eurex T7 Order Entry |
