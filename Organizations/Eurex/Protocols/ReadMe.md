## Eurex Protocols

Protocol specifications for Eurex Exchange and Deutsche Boerse T7 trading platform interfaces covering derivatives, cash equities, and clearing.

| Protocol | Type | Description |
| --- | --- | --- |
| [Eobi](Eobi.md) | Market Data | Enhanced Order Book Interface for T7 market data |
| [Eti](Eti.md) | Order Entry | Enhanced Trading Interface for T7 order entry |
| [Edci](Edci.md) | Market Data | Enhanced Derivatives Clearing Interface |

## Specifications

Protocol schema definitions for each interface are available in the [Specifications](../Specifications) directory.
