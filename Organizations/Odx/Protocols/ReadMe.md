## Odx Protocols

Protocol specifications for Osaka Digital Exchange Pts market data and order entry. Odx operates proprietary trading systems for Japanese equities and security tokens.

| Protocol | Type | Description |
| --- | --- | --- |
| [Equities Itch](Equities.Itch.md) | Market Data | Pts equities Itch market data |
| [SecurityToken Itch](SecurityToken.Itch.md) | Market Data | Pts security token Itch market data |
