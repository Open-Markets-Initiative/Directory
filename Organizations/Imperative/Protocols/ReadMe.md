## Imperative Protocols

Protocol specifications for Imperative Execution market data dissemination. Imperative operates the IntelligentCross venue, an alternative trading system using machine learning-based matching.

| Protocol | Type | Description |
| --- | --- | --- |
| [Mdf](Mdf.md) | Market Data | IntelligentCross market data feed |
