## Intelligent Cross


Intelligent Cross is Imperative Execution's SEC registered alternative trading system for US equities, using AI-optimized matching to reduce market impact. Publishes the Aspen-encoded Market Data Feed multicast protocol.


### Protocols


| Protocol | Type | Note |
| --- | --- | --- |
| [DepthOfBook](DepthOfBook.md) | MarketData | Imperative Intelligent Cross Equities Market Data |
