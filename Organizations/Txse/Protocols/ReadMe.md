## Txse Protocols

Protocol specifications for the Texas Stock Exchange (Txse) including order entry, market data, and session transport protocols.

| Protocol | Type | Description |
| --- | --- | --- |
| [Seed](Seed.md) | Order Entry | Binary order entry |
| [Feed](Feed.md) | Market Data | Binary market data |
| [Rake](Rake.md) | Transport | Session and framing layer |

## Specifications

Protocol schema definitions are available in the [Specifications](../Specifications) directory.
