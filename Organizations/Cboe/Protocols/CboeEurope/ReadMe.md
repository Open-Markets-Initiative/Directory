## Cboe Europe


Cboe Europe pan European equities exchange headquartered in Amsterdam operating across multiple European venues.

| Protocol | Type | Note |
| --- | --- | --- |
| [AuctionFeed](AuctionFeed.md) | MarketData | Cboe Europe Auction Event Data |
| [BinaryOrderEntry](BinaryOrderEntry.md) | OrderEntry | Cboe Europe Binary Order Entry |

| [TrfOrderEntry](TrfOrderEntry.md) | OrderEntry | Cboe Europe Trf Trade Report Submission |
