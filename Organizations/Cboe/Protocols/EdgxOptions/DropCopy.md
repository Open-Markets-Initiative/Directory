## EdgxOptions Drop Copy: Cboe Edgx Options Fix drop copy

Financial Information eXchange (Fix) drop copy protocol for Cboe Edgx Options.

### Overview

The Fix Drop port delivers copies of execution reports for orders entered on Cboe Edgx Options.

### Transport

Tcp Fix session per member with sequence tracking, heartbeats, and resend-based recovery.

### Key Characteristics

- **Fix** - Tag-value drop copy port
