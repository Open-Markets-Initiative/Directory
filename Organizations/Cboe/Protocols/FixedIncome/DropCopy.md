## FixedIncome Drop Copy: Cboe Fixed Income Fix drop copy

Financial Information eXchange (Fix) drop copy protocol for Cboe Fixed Income.

### Overview

The Fix Drop port delivers copies of execution reports for orders entered on Cboe Fixed Income.

### Transport

Tcp Fix session per member with sequence tracking, heartbeats, and resend-based recovery.

### Key Characteristics

- **Fix** - Tag-value drop copy port
