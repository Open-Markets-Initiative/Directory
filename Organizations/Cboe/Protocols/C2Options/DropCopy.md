## C2Options Drop Copy: Cboe C2 Options Fix drop copy

Financial Information eXchange (Fix) drop copy protocol for Cboe C2 Options.

### Overview

The Fix Drop port delivers copies of execution reports for orders entered on Cboe C2 Options.

### Transport

Tcp Fix session per member with sequence tracking, heartbeats, and resend-based recovery.

### Key Characteristics

- **Fix** - Tag-value drop copy port
