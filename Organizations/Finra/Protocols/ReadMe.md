## Finra Protocols

Protocol specifications for Finra market data dissemination services. All protocols use MoldUdp64 transport for multicast delivery of Otc equity, Otcbb, and agency debt trade data.

| Protocol | Type | Description |
| --- | --- | --- |
| [Tdds](Tdds.md) | Market Data | Trade Data Dissemination Service |
| [Bbds](Bbds.md) | Market Data | Bulletin Board Dissemination Service |
| [Atds](Atds.md) | Market Data | Agency Debt Trade Dissemination Service |

## Specifications

Protocol specifications for each protocol are available in the [Specifications](../Specifications) directory.
