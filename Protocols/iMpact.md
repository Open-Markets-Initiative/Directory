## iMpact: Ice iMpact compact fixed-width binary message encoding used by the Mdf market data feed

Ice's proprietary fixed-width binary message encoding for low-latency market data. Used as the wire format for the Multicast Data Feed (Mdf) carrying order book, trade, and reference data over Udp multicast with companion Tcp replay and snapshot services. Distinct from the Sbe encoding used by the Binary Order Gateway (Bgw) order entry protocol.
